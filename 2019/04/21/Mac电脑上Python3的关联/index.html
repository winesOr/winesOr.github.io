Mac电脑上Python3的关联

发表于 2019-04-21 更新于 2020-03-12 分类于 技术
本文字数： 261 阅读时长 ≈ 1 分钟
Mac电脑上安装Python3.x后，如何将系统默认的Python命令替换成Python3

查找Python和Python3的位置
which python
which python3
取消原有链接
unlink /usr/local/bin/python
注意：这里的位置是你自己电脑上安装Python的位置，一般默认就是这个
把链接改为Python3
ln -s /usr/local/bin/python3 /usr/local/bin/python

同理，查找pip和pip3的位置
which pip
which pip3
更改连接地址
ln -s /usr/local/bin/pip3 /usr/local/bin/pip
OK，到这里就大功告成了
